HEXO+GitHub搭建个人博客

🐲一直以来想搭建一个自己的博客，最近查了网上推荐使用的Hexo+GitHub的方式，自己尝试搭建一下我的个人博客，折腾了好半天，总算是搞出点小成果了。😃😃😃

下面就说一下自己搭建的过程以及一些问题，具体细节百度很多教程，主要说明需要重点注意的地方（❗️❗️❗️）。

尤其推荐Hexo官网教程👇

Hexo

¶环境配置

  1. 注册GitHub账号，在自己账户中新建一个库用于关联博客。❗️ ❗️ ❗️ 这里一定要注意，新建库的命必须为：GitHub用户名.github.io。
